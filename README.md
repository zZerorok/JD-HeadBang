# JD-HeadBang
4조

## 🖥️ 프로젝트 소개
앨범샵으로 유저가 로그인을 해서 가수의 이름이나 노래이름으로 검색해서 구매하는 프로그램입니다
<br>

## 🕰️ 개발 기간
* 24.04.26일 - 24.04.30일

### 🧑‍🤝‍🧑 맴버구성
 - 팀원1 : 김다빈 -
 - 팀원2 : 노영록 - 
 - 팀원3 : 송재혁 - 로그인입력과 MenuHandler와 view 부분의 출력 및 노래재생 부분을 담당함 
 - 팀원4 : 육슬찬 - 
 - 팀원5 : 이창연 -

### ⚙️ 개발 환경
-  ``` Java 17 ```
- ``` JDK-17.0.10+7 ```
- **IDE** : IntelliJ IDEA Ultimate 2023.3.6
